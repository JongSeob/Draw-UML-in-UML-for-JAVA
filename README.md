UML Diagram들을 Code로 그릴 수 있도록 하는 오픈소스 프로젝트.

**[PlantUML official website](https://plantuml.com/ko/)**

## PlantUML 예시 - Class Diagram

### Code

---

```
@startuml
class a {
}
class b {
}
a <|- b
@enduml
```
